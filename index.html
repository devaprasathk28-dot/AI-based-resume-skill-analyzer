How This Tool Helps You

This mini project simulates how Applicant Tracking Systems (ATS) and recruiters scan resumes for important keywords. It compares your resume against the skill-set needed for a particular role and shows a match score.

Role-Aware
Each role has its own expected skill set (Python Dev, Data Analyst, ML Engineer, etc.).
Soft Skills
Also checks for communication, teamwork, leadership and other soft skills.
Viva-Friendly
Easy to explain: Python, Flask, file handling, simple NLP-style matching.
